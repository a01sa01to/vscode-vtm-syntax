# Virtual Turing Machine Syntax

This is a syntax highlighter for the Virtual Turing Machine by stealthmate ([stealthmate/turing-machine](https://github.com/stealthmate/turing-machine)).

## Features

- Syntax Highlighting
- Linting
- Suggestions
